API 키 발급

자동화 도구 및 AI 연동을 위한 전용 열쇠 발급 가이드

⚠️ 보안 주의사항

발급된 API 키는 본인만이 알아야 하는 중요한 정보입니다. 타인에게 노출되면 본인의 할당량이 소모되거나 오용될 수 있으므로, 절대로 공개된 장소에 게시하지 마세요.

1 Google AI Studio 접속

먼저 Google AI Studio 사이트에 접속하여 구글 계정으로 로그인합니다.

2 API 키 만들기 클릭

화면 왼쪽 메뉴 또는 중앙에 위치한 파란색 [Create API key] (또는 API 키 만들기) 버튼을 클릭합니다.

3 프로젝트 선택 및 키 복사

프로젝트를 선택하여 키를 생성하면 팝업창에 API 키가 나타납니다. [Copy] 버튼을 눌러 키를 복사한 후, 사용하려는 웹앱이나 프로그램의 API 키 입력란에 붙여넣으세요.

+ (추가 설정) 프로젝트 관리

선택 사항

이 과정은 필수 사항이 아닙니다 (안 하셔도 됩니다). 더 세밀한 API 설정이나 프로젝트 관리가 필요한 경우 Google Cloud Console에서 새 프로젝트를 추가하거나 관리할 수 있습니다.
