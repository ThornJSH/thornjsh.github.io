# 가계도 그리기 (Genogram) 사용자 매뉴얼

사회복지 실천 현장에서 클라이언트의 가족 관계를 시각화하는 가계도 제작 도구입니다. 이 매뉴얼은 Windows 실행 프로그램(.exe) 버전을 기준으로 작성되었습니다.

---

## 🚀 최신 버전 정보: v1.4.4
v1.4.4 버전은 실무적인 편의 기능을 강화하고 심리적 관계를 시각화할 수 있는 기능이 추가되었습니다.

### 1. 주요 업데이트 내용
- **심리적 관계선 지원**: 가족 구성원 간의 **친밀, 소원, 갈등** 등 심리적 관계를 선으로 연결하여 정교하게 분석할 수 있습니다.
- **태아(Fetus) 추가**: 성별 선택 항목에 '태아'가 추가되어, 임신 중인 상태를 표준 기호로 표현할 수 있습니다.
- **자유 텍스트 입력**: 화면의 원하는 위치에 자유롭게 메모나 설명을 추가할 수 있어 가계도 사정 기능을 보완합니다.

### 2. 사용 방법 (핵심 기능)
- **인물 추가**: 성별(남성, 여성, 태아)을 선택하고 인물을 배치합니다.
- **관계 연결**: 두 인물을 선택하고 관계 유형(부부, 자녀, 심리적 관계 등)을 설정합니다.
- **텍스트 삽입**: '텍스트 추가' 기능을 활성화한 후 원하는 위치를 클릭하여 메모를 작성합니다.

## 💻 PC 버전 공통 특징 및 주의사항
설치 없이 바로 실행 가능한 독립형 소프트웨어입니다.

- **무설치 실행**: 별도의 설치 과정 없이 다운로드 후 바로 실행 가능합니다.
- **오프라인 지원**: 인터넷 연결이 없어도 데이터 저장 및 불러오기가 가능하여 보안이 중요한 현장에서 유용합니다.
- **데이터 관리 (familytree.db)**: 
  - 프로그램 실행 시 자동으로 `familytree.db` 파일이 생성됩니다.
  - 이 파일에 모든 가계도 데이터가 저장되므로, **파일을 삭제하지 않도록 주의**해 주세요.
- **제작 환경**: Python & PyQt6 기반으로 제작되었습니다 (Vibe Coding).
