1. 진행하기 전에.. (설정 필수)

노트북LM의 데이터를 Gemini가 읽어들일 수 있도록 설정을 해주셔야 합니다.

  • 📍 왼쪽 하단에 메뉴가 있습니다.
  • ⚙️ 설정 및 도움말 > 연결된 앱 > Google Workspace "연결"

2. 데이터 준비

PPT의 근거가 될 모든 텍스트 자료를 NotebookLM에 먼저 업로드합니다.

📄 다양한 형식 지원
PDF, .txt, 구글 문서, 구글 프레젠테이션 등
🌐 외부 소스 활용
웹사이트 URL 주소 또는 직접 입력한 텍스트
🎯 할 일
자료를 모두 올려 '소스' 탭에 리스트를 확인하세요.

💡 NotebookLM에 정리된 데이터가 정확할수록 Gemini가 생성하는 슬라이드의 퀄리티가 높아집니다.

3. Gemini 슬라이드 제작 설정

슬라이드는 Gemini로 만드는데, 이때 3가지 설정을 하셔야 합니다.

① 노트북 추가
"+" 버튼 > NotebookLM 선택 > 데이터 저장 노트북 추가
② Canvas 선택
"도구" 버튼 클릭 > "Canvas"를 선택
③ 사고 모드 변경
"빠른 모드" → "사고 모드"로 전환

※ (옵션) 추가 reference가 있다면 "+"버튼의 "파일업로드"를 활용하세요.

4. 프롬프트 입력 예시

간단 프롬프트
전략 컨설팅 전문가로서 첨부된 NotebookLM 데이터를 분석하여, 현대적이고 전문적인 비즈니스 스타일의 슬라이드 구성을 제안해줘. 각 슬라이드에는 명확한 문장형 제목, 3~5개의 핵심 불렛 포인트, 발표자용 강조 인사이트를 포함해줘. 데이터에 최적화된 차트(폭포형, 파이 등)와 인포그래픽 배치를 구체적으로 묘사하고, 전체 내용을 한글로 작성해줘.
🚀 추천 프롬프트 (더 세밀한 설정) 클릭하여 펼치기
당신은 전략 컨설팅 펌의 프레젠테이션 디자이너이자 콘텐츠 전략가입니다. 전달하는 [데이터/파일/주제]를 바탕으로, 현대적이고 전문적인 Google Slide(슬라이드) 덱 구성을 제안해 주세요. ### 1. 디자인 가이드라인 - 톤앤매너: [예: 신뢰감을 주는 비즈니스 스타일 / 혁신적이고 미니멀한 테크 스타일] - 메인 컬러: 현대적 감각에 맞춘 [예: 딥 네이비와 포인트 오렌지] 조합 (또는 첨부 파일의 디자인 시스템 준수) - 배경: 가독성을 극대화한 깔끔한 레이아웃 ### 2. 슬라이드 구성 원칙 (각 슬라이드 공통) - Headline: 한눈에 핵심이 파악되는 명확한 문장형 제목 - Body: 3~5개의 핵심 포인트를 불렛 포인트로 정리 - Visual Asset: 슬라이드 성격에 맞는 시각적 요소(타임라인, 차트, 인포그래픽, 도표, 핵심 인용구)를 구체적으로 묘사하고 배치 제안 - Speaker Note: 발표자가 강조해야 할 핵심 인사이트 1줄 추가
[💎 유료 버전(Gemini Advanced) 전용 옵션] - 이미지 생성: 각 슬라이드의 맥락에 가장 적합한 고품질 비즈니스 이미지를 AI로 직접 생성하여 슬라이드 내에 적절히 배치해 주세요.
### 3. 출력 형식 - 슬라이드 번호순으로 정리하여 한글로 작성해 주세요. - 각 슬라이드에서 사용할 적절한 차트 종류(예: 폭포형 차트, 파이 차트 등)를 명시해 주세요. --- 내용/데이터: NotebookLM의 데이터를 참조하세요.

5. Slides로 내보내기 및 저장

Gemini가 만들어준 내용을 우측 상단 "Slides로 내보내기"를 눌러 내 구글 드라이브에 저장합니다.

내보내기가 완료되면 왼쪽 하단에 새 슬라이드 생성됨          Slides열기가 보입니다. 이걸 누릅니다.

(또는 내 구글 드라이브에서 직접 확인 가능합니다.)

6. 수정 및 파워포인트 변환

만들어진 슬라이드 초안을 수정/편집한 후 파워포인트 파일로 다운로드합니다.

파일 > 다운로드 > Microsoft PowerPoint (.pptx)

💡 문제해결 (Troubleshooting)

"저는 언어 모델일 뿐이라서 그것을 도와드릴 수가 없습니다."라고 Gemini가 대답하나요?

보통 이 경우 Gemini가 스스로의 버전을 착각하고 있을 때 발생합니다.
구글 계정의 언어 설정을 '영어'로 변경한 뒤 다시 시도해 보세요. 언어를 영어로 바꾸면 시스템이 최신 상태로 갱신되어 가능해지는 경우가 많습니다.

✅ 팁: 영어로 슬라이드가 생성되었다면, 작업 완료 후에 "방금 만든 내용을 한글로 다시 번역해줘"라고 요청하면 깔끔하게 해결됩니다!

※ 이는 구글의 배포방식 때문에 발생하는 것으로 곧 이 과정 없이도 가능해질 것입니다.

💡 핵심 요약

결론적으로 말해, 노트북LM의 데이터를 기반으로 Gemini가 구글 슬라이드 초안을 만들어주면, 그걸 PPT 형식으로 변환 다운로드해서 사용하는 방식입니다.
