SNS 홍보문 만들기 (SNS Promotion)

Gemini API를 활용해 SNS 포스팅 문구를 자동으로 만들어주는 웹앱입니다. 웹 페이지의 URL이나 텍스트를 입력하면 **인스타그램, 페이스북, X(구 트위터)** 등 채널별 특성에 맞춰 홍보 문구를 생성합니다.

📱

채널별 맞춤형 문구

해시태그, 말투, 길이를 각 SNS 플랫폼에 최적화하여 제안합니다.

🔗

URL 기반 생성

기사나 공지사항 주소만 넣으면 인공지능이 내용을 파악해 홍보물을 구성합니다.

포스팅 생성기 실행 → 사본 만들기(설정)

채널별 자동 문구 생성 화면

💡 활용 팁

개인화된 사용을 위해서는 Gemini API 키 발급이 필요합니다. [설정] 버튼 또는 [사본 만들기]를 통해 본인의 API 키를 입력하면 무제한으로 활용이 가능합니다.
