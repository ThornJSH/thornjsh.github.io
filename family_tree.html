가계도 그리기 (Genogram)

사회복지 실천 현장에서 클라이언트의 가족 관계를 시각화하는 가계도 제작 도구입니다. 사용 환경에 따라 Windows 실행 프로그램(.exe) 또는 웹앱(Web App) 중 선택하여 사용하실 수 있습니다.

도구 선택

PC용

Windows 실행 프로그램 (.exe) v1.4.4

설치 없이 바로 실행 가능한 Windows용 프로그램입니다. 구글 로그인이나 사본 만들기 등의 복잡한 과정 없이 오프라인 환경에서도 안정적으로 가계도를 그릴 수 있습니다. 최근 업데이트를 통해 관계선 표현, 태아 추가, 자유 텍스트 입력 등 실무적인 기능을 강화하였습니다.

특징 및 v1.4.4 업데이트

  • v1.4.4 신규 기능: 관계선(친밀/소원/갈등), 태아 성별 추가, 자유 텍스트 메모 기능
  • familytree.db: 실행 시 생성되는 DB 파일에 데이터가 저장됩니다. (파일 삭제 주의)
  • 오프라인 지원: 인터넷 연결 없이도 데이터 저장 및 불러오기 가능
  • 바이브 코딩 제작: Python & PyQt6 기반 독립형 소프트웨어

💡 v1.4.4 주요 기능 활용 가이드

  • 심리적 관계선: 인물 간 관계(친밀, 소원, 갈등 등)를 선택하여 선으로 연결할 수 있어 보다 정교한 가계도 분석이 가능합니다.
  • 태아 추가: 성별 선택 옵션에서 태아를 선택하여 임신 중임을 가계도에 공식 기호로 표현할 수 있습니다.
  • 자유 텍스트 입력: 특정 인물을 클릭하지 않고도 화면 어디든 자유롭게 메모나 설명을 추가할 수 있습니다.
v1.4.4 다운로드(.exe) 사용자 매뉴얼
웹용

가계도 그리기 웹앱 (Web App)

별도의 프로그램 다운로드 없이 웹 브라우저에서 바로 사용할 수 있는 도구입니다. 데이터를 구글 스프레드시트에 연동하여 관리할 수 있으며, 반응형 디자인으로 모바일에서도 확인이 가능합니다.

중요 사용 가이드

안전하고 지속적인 사용을 위해 다음 과정을 추천합니다.

  • 사본 만들기: 제공된 예시의 [파일 > 사본 만들기]를 통해 개인 계정으로 복사하세요.
  • 데이터 보안: 저장된 데이터는 선택 삭제를 통해 관리 가능하며, 개인 사본 사용 시 더욱 안전합니다.
  • Apps Script: 스크립트 내부 코드는 훼손되지 않도록 주의 바랍니다.
