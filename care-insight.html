주요 기능

📝

상담 기록 자동 요약

긴 상담 내용을 핵심 요약과 욕구 분류로 즉시 변환합니다. 객관적 기록 작성이 쉬워집니다.

📊

사례회의 보고서 생성

여러 건의 기록을 모아 사례회의에 즉시 활용 가능한 종합 보고서 초안을 만듭니다.

🔒

강력한 개인정보 보호

데이터를 AI로 전송하기 전 실명을 자동 마스킹하며, 모든 데이터는 PC에만 저장됩니다.

⚙️

맞춤형 AI 설정

본인의 Gemini API 키를 사용하여 무료 한도 내에서 자유롭게 성능을 조정할 수 있습니다.
