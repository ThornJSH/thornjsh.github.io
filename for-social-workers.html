사회복지사를 위한 도구

사회복지사를 위한 유용한 도구들을 안내합니다. 바이브(Vibe) 코딩을 통해 만들어진 다양한 윈도우용 애플리케이션(.exe)과 웹앱, 자동화 스크립트 등을 공유합니다. 현장 업무의 효율을 높이는 데 활용해보세요.

사용 전 확인사항

이곳에서 제공되는 웹앱과 스크립트를 원활하게 사용하기 위해서는 다음과 같은 사전 과정이 필요할 수 있습니다.

1

사용 권한 승인 (초기 설정)

구글 기반 스크립트 최초 실행 시 권한 승인이 필요합니다. 이는 본인 계정에서 한 번만 수행하면 됩니다.

2

사본 만들기 (개인화)

제공된 예시는 테스트용입니다. 반드시 [파일 > 사본 만들기]를 통해 본인의 구글 드라이브로 복사한 후 사용하세요.

3

Gemini API 발급 및 적용

AI 기능 활용을 위해 Google AI Studio에서 API 키를 발급받아야 합니다. API 키는 타인과 공유하지 마세요.
