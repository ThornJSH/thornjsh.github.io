엑셀 비밀번호 일괄 변경

개인정보 보호를 위한 필수 도구

개인정보가 포함된 경우 해당 문서에는 비밀번호를 설정해두어야 합니다. 만일 비밀번호를 설정하지 않았다면 일일이 해당 파일을 열어서 비밀번호를 설정해주어야 하는데, 이 과정이 귀찮기 그지 없습니다. 일괄 비밀번호 설정 프로그램으로 한 번에 해결하세요.

사용 방법

  1. 엑셀 파일이 저장되어 있는 폴더의 상위 폴더에 이 실행파일을 복사해 둡니다.
    ※ C:\ 등과 같은 최상위 폴더에서는 실행하지 마세요.
  2. 다운받은 ExcelPasswordChanger.exe를 실행합니다 (하위폴더까지 자동으로 검색합니다).
  3. 기존 비밀번호가 있는 경우 먼저 이를 입력합니다.
  4. 변경할 비밀번호를 입력합니다.
  5. 만일 기존의 비밀번호를 입력하고 새 비밀번호를 입력하지 않으면 비밀번호가 삭제됩니다.
  6. 오타가 걱정된다면 비밀번호 입력 칸 옆의 아이콘을 눌러 직접 확인해보세요.

참고: 비밀번호 설정 실패 이력은 엑셀 파일이 있는 폴더에 failure_log_날짜.txt로 저장됩니다. 파일이 열려있다거나, 기존에 비밀번호가 설정되어 있는 경우 "실패"하게 됩니다.
