# Smart Welfare Tech: 사회복지사를 위한 AI 활용 포털

사회복지 현장의 디지털 전환과 AI 기술 활용을 돕기 위한 전문 포털 사이트입니다. 
사회복지사가 실무에서 즉시 활용할 수 있는 다양한 자동화 도구, 가이드, 그리고 AI 프롬프트를 제공합니다.

🔗 **바로가기:** [https://thornjsh.github.io/](https://thornjsh.github.io/)

---

## 🚀 주요 기능 및 라이브러리

### 1. 스마트 복지기술 (Smart Welfare Tech)
- **구글 스프레드시트**: 데이터와 서식의 분리, 자동화의 기초 가이드 제공
- **노트북LM**: 전문 지식 베이스 구축 및 활용
- **AI 챗봇 만들기**: 기관 전용 AI 상담 도구 구축 가이드

### 2. 사회복지사를 위한 실무 도구 (For Social Workers)
- **가계도/생태도 그리기**: 복잡한 사정 도구를 쉽고 빠르게 시각화
- **PDF 플립북**: 홍보물이나 보고서를 입체적인 플립북으로 변환
- **표 스타일 정리 도구**: 한글/워드 문서를 위한 깔끔한 표 스타일링 지원

### 3. AI와 함께하는 협업 (With AI)
- **AI Prompt Pro**: 각 상황별 최적화된 프롬프트 제공
- **SNS 홍보문 만들기**: 행사 및 프로그램 홍보를 위한 맞춤형 문구 생성
- **인권 지향적 글쓰기**: AI를 활용한 민감성 있는 문서 검토
- **Docs/Sheet/Slide AI**: 구글 워크스페이스 내 AI 통합 활용법

### 4. 미니 앱 및 유틸리티 (Mini Apps)
- **윈도우 시계 설정**: 업무 효율을 위한 OS 세부 설정
- **시간조건 파일 복사**: 특정 조건의 파일을 자동 분류 및 복제
- **PDF MultiTool**: PDF 분할, 합치기, 보안 설정 등

### 5. 통계 다루기 (Statistics)
- **기초 통계 분석**: 데이터의 기본 성질 파악
- **사전-사후 비교 분석**: 프로그램 효과성 검증을 위한 통계 도구
- **IPA 분석**: 중요도와 만족도를 바탕으로 한 전략 수립
