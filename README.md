# 사회복지사의 AI 활용 (AI for Social Workers)

이 프로젝트는 사회복지 현장에서의 디지털 전환과 AI 기술 활용을 돕기 위해 구축된 정보 공유 플랫폼입니다. 기존 Google Sites에서 구축되었던 내용을 정적 사이트로 변환하여 GitHub Pages를 통해 서비스하고 있습니다.

## 🚀 주요 기능 및 특징

- **스마트복지기술**: AI와 디지털 기술이 사회복지 현장에 가져오는 변화와 인재상 공유.
- **스마트워크**: 구글 시트, 문서 등 업무 효율화를 위한 도구 활용법 가이드.
- **AI 프롬프트**: 사회복지 업무에 최적화된 마크다운 기반 프롬프트 사례 및 이해.
- **통계 다루기**: 현장 데이터를 효율적으로 수집하고 분석하는 방법 안내.
- **사회복지용 미니 앱**: 업무 효율을 높여주는 전용 웹 도구 및 AI 서비스 링크 제공.

## 🛠 기술 스택 및 작업 내용

- **기반 기술**: HTML5, CSS3, JavaScript (Vanilla JS)
- **호스팅**: GitHub Pages
- **주요 수정 사항**: 
  - Google Sites 기반 레이아웃의 모바일 대응 (반응형 메뉴 최적화)
  - 부드러운 사이드바 인터페이스 및 하위 메뉴(Dropdown) 구현
  - 불필요한 위젯 제거 및 사용자 맞춤형 헤더 스타일링
  - 전역 스크립트를 통한 효율적인 메뉴 관리

## 📝 라이선스 및 문의

- **제작/배포**: [welfareact.net](https://welfareact.net)
- **문의**: 사이트 내 Q&A 또는 관련 소셜 채널을 통해 문의해 주세요.
