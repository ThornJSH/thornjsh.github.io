마크다운의 이해

마크다운(Markdown)

마크다운(Markdown)은 문서를 쉽고 간단하게 작성할 수 있도록 고안된 경량 마크업 언어입니다. 특수기호(#, *, >, 등)를 이용해 제목, 목록, 강조, 링크 같은 문서 구조와 서식을 표현합니다. 작성한 문서는 읽기 쉽고, HTML 등으로 변환이 가능하여 웹 문서 작성에 널리 활용됩니다.

AI에게 질문할 때 이런 마크다운을 활용하면 보다 구체적이고 직접적인 질문이 가능해 보다 좋은 결과물을 얻을 수 있습니다.

1. 제목 ( # )

# 기호를 사용해 제목을 만듭니다. # 개수가 많을수록 작은 수준의 제목이 됩니다.

2. 목록

가. 순서 없는 목록 ( - ) 나. 순서 있는 목록 ( 1. )

3. 강조 ( *, _ )

가. 굵게: **텍스트**, __텍스트__ 나. 기울임: *텍스트*, _텍스트_ 다. 굵고 기울임: ***텍스트***

4. 구분선 ( --- )

영역이 바뀌거나 내용이 분리되는 경우에는 구분선을 사용해 표시해줍니다.

면접 질문하기

역할

너는 15년차 이상의 경력을 가진 사회복지시설의 채용 팀장이야. 지원자가 제출한 자기소개서를 바탕으로 심층 면접 질문을 만드는 역할을 맡았어.

목표

자기소개서의 내용을 기반으로 지원자가 우리 시설에 적합한 인재인지 다각도로 검증하는 것이 목표야. 특히, 자기소개서에 기술된 경험과 성과가 사실인지 확인하고, 실제 행동·사고·가치관을 깊이 파악할 수 있는 질문을 만들어야 해.

평가 기준

아래 3가지 핵심 기준에 따라 지원자를 평가할 수 있는 질문을 생성해 줘. 각 기준별로 질문을 분류하고, 질문은 행동/경험 기반 질문(BEI) 형식으로 만들어야 해. 지원자가 단답형으로 끝낼 수 없도록, 구체적 사례와 행동을 설명하게 유도해 줘. 또한, 자기소개서의 진위 여부를 검증할 수 있는 질문도 포함해 줘.

1. 사회복지 가치 및 윤리 (철학) - 지원자가 사회복지사로서의 소명의식과 가치관을 진정으로 내재화하고 있는지 확인. - 자기소개서에 드러난 가치관이 실제 경험과 어떻게 연결되는지 질문. - 딜레마 상황에서의 윤리적 판단 능력을 묻는 질문 포함.

2. 사회복지 실무 역량 (기술) - 자기소개서에 작성된 경험(실습, 봉사, 경력 등)의 사실 여부와 성과를 구체적으로 확인. - 언급된 역량(예: 사례관리, 프로포절 작성, 상담 기법 등)을 실제 업무에서 어떻게 발휘할 수 있을지 질문. - 경험을 통해 무엇을 배우고 느꼈는지(성찰)를 묻는 질문 포함. - 자기소개서에서 언급한 기관명·활동명·성과 등을 추가 확인할 수 있는 질문 포함.

3. 직장인으로서의 기본 자질 (태도) - 조직 내에서 동료들과 협력하고 소통하는 능력. - 업무 중 갈등이나 스트레스 상황에 대처하는 방식. - 본인의 부족한 점을 인지하고 성장하려는 의지가 있는지 확인. - 자기소개서에 언급된 대인관계 경험이나 성장 경험이 실제 있었는지 확인하는 질문 추가.

질문 형식

- 각 평가 기준별로 3~4개의 주요 질문을 만들어 줘. - 각 주요 질문에는 지원자의 답변에 따라 추가적으로 질문할 수 있는 ##꼬리 질문(Follow-up question)##을 1~2개씩 포함해 줘. - 질문은 지원자가 경험·사례·행동을 구체적으로 말하게 유도해야 해. - 각 질문에는 자기소개서 진위 여부를 확인할 수 있는 요소를 하나 이상 포함해 줘.
