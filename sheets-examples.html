DX 스프레드시트 수식 활용 예제

실무에서 바로 활용 가능한 다양한 수식과 템플릿을 소개합니다.

어려운 수식을 외우지 않아도 괜찮습니다. 아래 예제들을 통해 원리를 이해하고, 필요한 시트의 사본을 만들어 본인의 업무에 맞춰 수정해 보세요.

📊 사회복지시설 종사자 경력인정 계산기

여러 기관에서의 근무 경력을 합산하고, 지침에 따른 인정 비율(100%, 80% 등)을 적용하여 현재 시점의 총 경력을 연, 월, 일 단위로 정확하게 산출합니다.

  • 실시간 경력 갱신: 주황색 셀(오늘 날짜)을 기준으로 재직 중인 경력이 매일 자동 업데이트됩니다.
  • 정확한 합산 로직: 30일을 1개월로, 12개월을 1년으로 환산하는 표준 방식을 따릅니다.
  • 수식 보호 주의: 정확한 날짜 계산(초일 산입 등)을 위한 '조정1, 조정2' 열은 절대 수정하지 마세요.
  • 인정 비율 설정: 입사/퇴사일과 함께 각 경력별 인정 비율만 입력하면 모든 계산이 끝납니다.

활용된 핵심 함수

🗓️ 주간업무일지 및 구글 캘린더 연동

개별 이름 시트에 직원이 일시와 업무 내용을 입력하면, 구글 앱스 스크립트(GAS)를 통해 지정된 구글 캘린더로 업무 일정을 자동 전송하고 팀원과 공유하는 시스템입니다.

  • 개별 시트 입력: 직원별 시트에 작성된 일정이 통합 관리됩니다.
  • 캘린더 자동 전송: 입력된 데이터가 실시간 또는 주기적으로 캘린더에 연동됩니다.
  • 맞춤형 ID 설정: Apps Script 코드 내에서 내부소통용 및 사업공유용 캘린더 ID를 손쉽게 변경할 수 있습니다.
  • 자동화 트리거: 수동 실행 없이도 일정 주기에 맞춰 데이터를 동기화합니다.

⏰ 근태관리대장 및 연차 자동 계산

직원별 출퇴근 기록을 바탕으로 근무 시간, 연장 근로, 잔여 연차 등을 자동으로 관리하는 템플릿입니다.

📍 GPS 위치 정보를 활용한 스마트 출근부

스마트폰의 GPS 정보를 활용하여 지정된 근무지 반경 내에서만 원터치로 출퇴근을 기록하고, 이를 구급 스프레드시트에 실시간으로 저장하는 시스템입니다.

  • 정교한 위치 검증: 설정된 위경도 좌표 기준 100m 이내에서만 기록 가능
  • 보안 수준 강화: 30초 유효 토큰(OTT) 및 5분 이내 중복 기록 차단
  • 다양한 모드 지원: QR코드 전용 uid 접속, 자동 출퇴근 판단 모드 등
  • 다중 근무지 관리: 한 명의 사용자가 여러 장소에 등록 시 자동 매칭

📅 사회복지시설 종사자 호봉 관리 시스템

사회복지시설 종사자의 기준정보를 바탕으로 연도별 획정 호봉을 보여주는 템플릿입니다.

  • 승급월: 직접 입력합니다. 자동으로 입력하려면 아래 수식을 사용하세요.
    =IF(DAY(D2)=1, MONTH(D2), MONTH(EDATE(D2, 1)))
  • 기준호봉: 기존 경력(군경력 포함)이 인정된다면, 해당경력을 입력해줍니다.
  • 조건부 서식: 퇴사 발생시 해당 당사자의 개인정보를 보이지 않도록 처리합니다.

🏖️ 연가 관리 및 촉진일 자동 계산

A~F열에 기본정보를 입력하면 자동으로 연가일수와 연가촉진일을 계산해주는 시트입니다.

  • 간편한 설정: A1셀에 해당 연도(숫자)만 입력하면 당해연도 정보가 자동 갱신됩니다.
  • 자동 계산: F3~N3 셀은 Arrayformula로 처리되어 별도 조치 없이 자동 계산됩니다. (수정 금지)
  • 연가 촉진일: 근로기준법에 따른 연가 사용 촉진 시기를 자동으로 알려줍니다.
  • 조건부 서식: 퇴사일을 입력하면 해당 당사자의 개인정보를 보이지 않도록 처리합니다.
