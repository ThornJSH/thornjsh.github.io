생태도 그리기 (Ecomap)

클라이언트를 둘러싼 환경과 상호작용을 한눈에 파악할 수 있는 생태도 제작 도구입니다. Windows용 프로그램(.exe)과 웹앱(Web App) 중 선호하는 방식을 선택하여 사용하세요.

도구 선택

PC용

Windows 실행 프로그램 (.exe)

설치 없이 바로 실행 가능한 Windows 전용 프로그램입니다. 구글 로그인 없이도 PC에 데이터를 안전하게 보관하며 작업할 수 있어 보안이 중요한 환경에서 권장됩니다.

특징 및 주의사항

  • ecomap_local.db: 데이터 저장을 위해 자동으로 생성되는 DB 파일입니다. (삭제 주의)
  • 설치 불필요: 별도의 설치 과정 없이 실행 파일만으로 작동
  • 안정적 환경: 인터넷이 연결되지 않은 곳에서도 로컬 데이터 작업 가능
다운로드 (.exe) 사용자 매뉴얼
웹용

생태도 그리기 웹앱 (Web App)

웹 브라우저를 통해 어디서나 즉시 사용할 수 있는 도구입니다. 간단한 입력만으로 생태도를 생성하며, 구글 스프레드시트와의 연동을 통해 데이터를 체계적으로 관리할 수 있습니다.

사용 권장사항

  • 사본 만들기: 구글 계정으로 [파일 > 사본 만들기] 후 개인 환경에서 사용하세요.
  • 데이터 보안: 사용 후 데이터 삭제 기능을 활용하여 보안을 유지하세요.
  • 반응형 디자인: 태블릿 등 다양한 기기에서 확인 가능 (v2.3)
